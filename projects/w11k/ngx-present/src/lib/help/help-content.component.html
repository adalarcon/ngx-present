Navigation

menu Click on the menu icon in the upper left corner to open the menu.
Via the menu you can navigate to the different modes like slide-by-slide, overview and presenter. You can also use the table of content to jump to specific sections, if the author of the presentation uses this feature.

Navigation in Slide Mode

Key Action
Arrow Right Go to next slide
Arrow Down Go to next slide
Page Down Go to next slide
Arrow Left Go to previous slide
Arrow Up Go to previous slide
Page Up Go to previous slide
ALT + Arrow Right Go to next table-of-content entry
ALT + Arrow Down Go to next table-of-content entry
ALT + Arrow Left Go to previous table-of-content entry
ALT + Arrow Up Go to previous table-of-content entry
Pos 1 Go to first slide
m Open menu
o Go to table of content
